# Get Order

Retrieve detailed information about a specific order.

<Note>
  Requires `read:orders` scope.
</Note>

## Request

```bash
GET /api/v1/orders/{id}
```

### Path Parameters

<ParamField path="id" type="integer" required>
  The order ID
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  The order object with full details including line items
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X GET "https://api.zenflow.com/api/v1/orders/12345" \
  -H "X-API-Key: zenflow_live_your_key"
```

```javascript Node.js
const response = await fetch(
  'https://api.zenflow.com/api/v1/orders/12345',
  {
    headers: { 'X-API-Key': 'zenflow_live_your_key' }
  }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.zenflow.com/api/v1/orders/12345',
    headers={'X-API-Key': 'zenflow_live_your_key'}
)
data = response.json()
```

</CodeGroup>

## Error Responses

### 404 Not Found

```json
{
  "success": false,
  "error": {
    "code": "not_found",
    "message": "Order not found"
  }
}
```

### 400 Bad Request

```json
{
  "success": false,
  "error": {
    "code": "invalid_id",
    "message": "Order ID must be a valid integer"
```
