# Get Order Status

Retrieve the current status of an order.

<Note>
  Requires `read:orders` scope.
</Note>

## Request

```bash
GET /api/v1/orders/{id}/status
```

### Path Parameters

<ParamField path="id" type="integer" required>
  The order ID
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="order_id" type="integer">
      Internal order ID
    </ResponseField>
    <ResponseField name="external_id" type="string">
      Your order ID (order_tenant_id)
    </ResponseField>
    <ResponseField name="state_id" type="integer">
      Current state ID
    </ResponseField>
    <ResponseField name="state_picking_id" type="integer">
      Picking state ID (if in picking flow)
    </ResponseField>
    <ResponseField name="updated_at" type="string">
      Last update timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X GET "https://api.zenflow.com/api/v1/orders/12345/status" \
  -H "X-API-Key: zenflow_live_your_key"
```

```javascript Node.js
const response = await fetch(
  'https://api.zenflow.com/api/v1/orders/12345/status',
  {
    headers: { 'X-API-Key': 'zenflow_live_your_key' }
  }

const data = await response.json();
```

```python Python

response = requests.get(

```

## Order States

| State ID | Name | Description |
|----------|------|-------------|
| 1 | Pending | Order received, awaiting processing |
| 2 | In Progress | Order is being picked |
| 3 | Packed | Order is packed and ready |
| 4 | Completed | Order shipped |
| 5 | Cancelled | Order was cancelled |
