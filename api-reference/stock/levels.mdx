# Get Stock Levels

Retrieve stock levels across all products and locations.

<Note>
  Requires `read:stock` scope.
</Note>

## Request

```bash
GET /api/v1/stock
```

### Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Items per page (max 100)
</ParamField>

<ParamField query="product_id" type="integer">
  Filter by product ID
</ParamField>

<ParamField query="location_id" type="string">
  Filter by location ID
</ParamField>

<ParamField query="low_stock" type="boolean">
  Only show items below minimum threshold
</ParamField>

<ParamField query="search" type="string">
  Search by product name, barcode, or SKU
</ParamField>

## Example

```bash
curl -X GET "https://api.zenflow.com/api/v1/stock?low_stock=true" \
  -H "X-API-Key: zenflow_live_your_key"
```
