# Get Product Stock

Retrieve stock levels for a specific product across all locations.

<Note>
  Requires `read:stock` scope.
</Note>

## Request

```bash
GET /api/v1/stock/product/{productId}
```

### Path Parameters

<ParamField path="productId" type="integer" required>
  The product ID
</ParamField>

## Example

```bash
curl -X GET "https://api.zenflow.com/api/v1/stock/product/100" \
  -H "X-API-Key: zenflow_live_your_key"
```
