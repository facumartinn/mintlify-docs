# Get Stock Movements

Retrieve the history of stock movements (additions, removals, transfers).

<Note>
  Requires `read:stock` scope.
</Note>

## Request

```bash
GET /api/v1/stock/movements
```

### Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Items per page (max 100)
</ParamField>

<ParamField query="product_id" type="integer">
  Filter by product ID
</ParamField>

<ParamField query="location_id" type="string">
  Filter by location ID
</ParamField>

<ParamField query="movement_type" type="string">
  Filter by movement type (IN, OUT, TRANSFER, ADJUSTMENT)
</ParamField>

<ParamField query="start_date" type="string">
  Filter from date (YYYY-MM-DD)
</ParamField>

<ParamField query="end_date" type="string">
  Filter to date (YYYY-MM-DD)
</ParamField>

## Example

```bash
curl -X GET "https://api.zenflow.com/api/v1/stock/movements?product_id=100&start_date=2024-01-01" \
  -H "X-API-Key: zenflow_live_your_key"
```

## Movement Types

| Type | Description |
|------|-------------|
| `IN` | Stock added (receiving, returns) |
| `OUT` | Stock removed (orders, damages) |
| `TRANSFER` | Stock moved between locations |
| `ADJUSTMENT` | Inventory count correction |
