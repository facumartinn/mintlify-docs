# Get Webhook

Retrieve detailed information about a specific webhook.

<Note>
  Requires `read:webhooks` scope.
</Note>

## Request

```bash
GET /api/v1/webhooks/{id}
```

### Path Parameters

<ParamField path="id" type="string" required>
  The webhook ID
</ParamField>

## Example

```bash
curl -X GET "https://api.zenflow.com/api/v1/webhooks/wh_abc123" \
  -H "X-API-Key: zenflow_live_your_key"
```
