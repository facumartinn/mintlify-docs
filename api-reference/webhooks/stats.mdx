# Get Delivery Statistics

Retrieve delivery statistics for a webhook.

<Note>
  Requires `read:webhooks` scope.
</Note>

## Request

```bash
GET /api/v1/webhooks/{id}/stats
```

### Path Parameters

<ParamField path="id" type="string" required>
  The webhook ID
</ParamField>

### Query Parameters

<ParamField query="days" type="integer" default="30">
  Number of days to include in statistics
</ParamField>

## Example

```bash
curl -X GET "https://api.zenflow.com/api/v1/webhooks/wh_abc123/stats?days=7" \
  -H "X-API-Key: zenflow_live_your_key"
```

### Response

```json
{
  "success": true,
  "data": {
    "total": 250,
    "success": 245,
    "failed": 3,
    "pending": 2,
    "avg_response_time_ms": 185,
    "success_rate": 98.0
  }
}
```

## Fields

| Field | Description |
|-------|-------------|
| `total` | Total delivery attempts in period |
| `success` | Successful deliveries |
| `failed` | Failed deliveries (after all retries) |
| `pending` | Deliveries in progress or awaiting retry |
| `avg_response_time_ms` | Average response time |
| `success_rate` | Success percentage (0-100) |
