# API de Webhooks

Los webhooks te permiten recibir notificaciones HTTP en tiempo real cuando ocurren eventos en ZenFlow.

<Note>
  Para una guía completa sobre implementación de webhooks, consulta la [Guía de Webhooks](/guides/webhooks).
</Note>

## Endpoints Disponibles

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| GET | `/webhooks/events` | Listar eventos disponibles |
| GET | `/webhooks` | Listar webhooks |
| GET | `/webhooks/:id` | Obtener detalles del webhook |
| POST | `/webhooks` | Crear webhook |
| PUT | `/webhooks/:id` | Actualizar webhook |
| DELETE | `/webhooks/:id` | Eliminar webhook |
| POST | `/webhooks/:id/rotate-secret` | Rotar secret |
| POST | `/webhooks/:id/toggle` | Habilitar/deshabilitar |
| GET | `/webhooks/:id/deliveries` | Historial de entregas |
| GET | `/webhooks/:id/stats` | Estadísticas |
| POST | `/webhooks/:id/test` | Enviar evento de prueba |

## Tipos de Eventos

| Evento | Descripción |
|--------|-------------|
| `order.created` | Nuevo pedido creado |
| `order.updated` | Pedido actualizado |
| `order.cancelled` | Pedido cancelado |
| `order.completed` | Pedido completado |
| `product.created` | Producto creado |
| `product.updated` | Producto actualizado |
| `product.deleted` | Producto eliminado |
| `stock.updated` | Nivel de stock modificado |
| `stock.low_alert` | Stock por debajo del mínimo |
| `stock.movement_created` | Movimiento de stock registrado |
| `flow.started` | Flujo de picking iniciado |
| `flow.completed` | Flujo de picking completado |
| `flow.cancelled` | Flujo de picking cancelado |

## Payload del Webhook

Todos los payloads de webhook siguen esta estructura:

```json
{
  "id": "evt_abc123",
  "event": "order.created",
  "created_at": "2024-01-15T10:30:00Z",
  "data": {
    // Datos específicos del evento
  }
}
```

## Seguridad

Los webhooks están firmados usando HMAC-SHA256. Verifica la firma en el header `X-Webhook-Signature`:

```
t=1705312200,v1=5d4b8c7a...
```

Consulta la [Guía de Webhooks](/guides/webhooks#verificando-webhooks) para ejemplos de código de verificación.
