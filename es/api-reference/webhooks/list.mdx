# Listar Webhooks

Obtén todos los webhooks configurados.

<Note>Requiere scope `read:webhooks`.</Note>

## Solicitud

```bash
GET /api/v1/webhooks
```

### Parámetros de Consulta

<ParamField query="active" type="boolean">
  Filtrar por estado activo
</ParamField>

## Ejemplo

```bash
curl -X GET "https://api.zenflow.com.ar/api/v1/webhooks" \
  -H "X-API-Key: zenflow_live_your_key"
```
