# Obtener Historial de Entregas

Obtén el historial de entregas de un webhook.

<Note>Requiere scope `read:webhooks`.</Note>

## Solicitud

```bash
GET /api/v1/webhooks/{id}/deliveries
```

### Parámetros de Ruta

<ParamField path="id" type="string" required>
  El ID del webhook
</ParamField>

### Parámetros de Consulta

<ParamField query="page" type="integer" default="1">
  Número de página
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Items por página
</ParamField>

<ParamField query="status" type="string">
  Filtrar por estado: `pending`, `success`, `failed`, `retrying`
</ParamField>

<ParamField query="event_type" type="string">
  Filtrar por tipo de evento
</ParamField>

<ParamField query="start_date" type="string">
  Filtrar desde fecha
</ParamField>

<ParamField query="end_date" type="string">
  Filtrar hasta fecha
</ParamField>

## Ejemplo

```bash
curl -X GET "https://api.zenflow.com.ar/api/v1/webhooks/wh_abc123/deliveries?status=failed" \
  -H "X-API-Key: zenflow_live_your_key"
```

## Estados de Entrega

| Estado     | Descripción                              |
| ---------- | ---------------------------------------- |
| `pending`  | Esperando entrega                        |
| `success`  | Entregado exitosamente (respuesta 2xx)   |
| `failed`   | Todos los intentos de reintento agotados |
| `retrying` | Programado para reintento                |
