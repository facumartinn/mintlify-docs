# Obtener Webhook

Obtén información detallada sobre un webhook específico.

<Note>Requiere scope `read:webhooks`.</Note>

## Solicitud

```bash
GET /api/v1/webhooks/{id}
```

### Parámetros de Ruta

<ParamField path="id" type="string" required>
  El ID del webhook
</ParamField>

## Ejemplo

```bash
curl -X GET "https://api.zenflow.com.ar/api/v1/webhooks/wh_abc123" \
  -H "X-API-Key: zenflow_live_your_key"
```
