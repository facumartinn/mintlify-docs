# Actualizar Webhook

Actualiza la configuración de un webhook existente.

<Note>Requiere scope `write:webhooks`.</Note>

## Solicitud

```bash
PUT /api/v1/webhooks/{id}
```

### Parámetros de Ruta

<ParamField path="id" type="string" required>
  El ID del webhook
</ParamField>

### Parámetros del Body

Todos los campos son opcionales. Solo incluye los campos que deseas actualizar.

<ParamField body="name" type="string">
  Nombre del webhook
</ParamField>

<ParamField body="url" type="string">
  URL del webhook
</ParamField>

<ParamField body="events" type="array">
  Tipos de eventos a suscribir
</ParamField>

<ParamField body="description" type="string">
  Descripción
</ParamField>

<ParamField body="headers" type="object">
  Headers personalizados
</ParamField>

<ParamField body="active" type="boolean">
  Habilitar/deshabilitar el webhook
</ParamField>

<ParamField body="timeout_ms" type="integer">
  Timeout de solicitud
</ParamField>

<ParamField body="retry_count" type="integer">
  Número de reintentos
</ParamField>

## Ejemplo

```bash
curl -X PUT "https://api.zenflow.com.ar/api/v1/webhooks/wh_abc123" \
  -H "X-API-Key: zenflow_live_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "events": ["order.created", "order.updated", "order.completed", "order.cancelled"],
    "timeout_ms": 60000
  }'
```

### Respuesta

```json
{
  "success": true,
  "data": {
    "id": "wh_abc123",
    "name": "Notificaciones de Pedidos",
    "url": "https://example.com/webhooks/zenflow",
    "events": [
      "order.created",
      "order.updated",
      "order.completed",
      "order.cancelled"
    ],
    "active": true,
    "timeout_ms": 60000,
    "updated_at": "2024-01-14T15:30:00Z"
  }
}
```
