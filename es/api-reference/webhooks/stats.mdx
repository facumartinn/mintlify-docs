# Obtener Estadísticas de Entregas

Obtén estadísticas de entregas de un webhook.

<Note>Requiere scope `read:webhooks`.</Note>

## Solicitud

```bash
GET /api/v1/webhooks/{id}/stats
```

### Parámetros de Ruta

<ParamField path="id" type="string" required>
  El ID del webhook
</ParamField>

### Parámetros de Consulta

<ParamField query="days" type="integer" default="30">
  Número de días a incluir en las estadísticas
</ParamField>

## Ejemplo

```bash
curl -X GET "https://api.zenflow.com.ar/api/v1/webhooks/wh_abc123/stats?days=7" \
  -H "X-API-Key: zenflow_live_your_key"
```

### Respuesta

```json
{
  "success": true,
  "data": {
    "total": 250,
    "success": 245,
    "failed": 3,
    "pending": 2,
    "avg_response_time_ms": 185,
    "success_rate": 98.0
  }
}
```

## Campos

| Campo                  | Descripción                                         |
| ---------------------- | --------------------------------------------------- |
| `total`                | Total de intentos de entrega en el período          |
| `success`              | Entregas exitosas                                   |
| `failed`               | Entregas fallidas (después de todos los reintentos) |
| `pending`              | Entregas en progreso o esperando reintento          |
| `avg_response_time_ms` | Tiempo de respuesta promedio                        |
| `success_rate`         | Porcentaje de éxito (0-100)                         |
