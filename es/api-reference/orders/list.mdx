# Listar Pedidos

Obtén una lista de pedidos de tu tenant con filtrado y paginación opcionales.

<Note>Requiere scope `read:orders`.</Note>

## Solicitud

```bash
GET /api/v1/orders
```

### Parámetros de Consulta

<ParamField query="page" type="integer" default="1">
  Número de página para paginación
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Número de pedidos por página (máx 100)
</ParamField>

<ParamField query="state_id" type="integer | integer[]">
  Filtrar por ID(s) de estado del pedido
</ParamField>

<ParamField query="start_date" type="string">
  Filtrar pedidos creados en o después de esta fecha (YYYY-MM-DD)
</ParamField>

<ParamField query="end_date" type="string">
  Filtrar pedidos creados en o antes de esta fecha (YYYY-MM-DD)
</ParamField>

<ParamField query="assembly_date" type="string">
  Filtrar por fecha de ensamblaje/envío (YYYY-MM-DD)
</ParamField>

<ParamField query="include_details" type="boolean" default="false">
  Incluir líneas de pedido en la respuesta
</ParamField>

## Respuesta

<ResponseField name="success" type="boolean">
  Si la solicitud fue exitosa
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="propiedades">
    <ResponseField name="orders" type="array">
      Array de objetos de pedido
    </ResponseField>
    <ResponseField name="pagination" type="object">
      <Expandable title="propiedades">
        <ResponseField name="total" type="integer">
          Número total de pedidos que coinciden con los filtros
        </ResponseField>
        <ResponseField name="page" type="integer">
          Número de página actual
        </ResponseField>
        <ResponseField name="limit" type="integer">
          Items por página
        </ResponseField>
        <ResponseField name="total_pages" type="integer">
          Número total de páginas
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Ejemplos

### Solicitud Básica

<CodeGroup>

```bash cURL
curl -X GET "https://api.zenflow.com.ar/api/v1/orders?page=1&limit=20" \
  -H "X-API-Key: zenflow_live_your_key"
```

```javascript Node.js
const response = await fetch(
  "https://api.zenflow.com.ar/api/v1/orders?page=1&limit=20",
  {
    headers: {
      "X-API-Key": "zenflow_live_your_key",
    },
  }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.zenflow.com.ar/api/v1/orders',
    params={'page': 1, 'limit': 20},
    headers={'X-API-Key': 'zenflow_live_your_key'}
)
data = response.json()
```

    "message": "La API key proporcionada es inválida"
