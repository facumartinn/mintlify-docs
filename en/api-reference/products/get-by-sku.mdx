# Get Product by SKU

Retrieve a product using its SKU (external_id).

<Note>Requires `read:products` scope.</Note>

## Request

```bash
GET /api/v1/products/sku/{sku}
```

### Path Parameters

<ParamField path="sku" type="string" required>
  The product SKU (external_id)
</ParamField>

## Example

```bash
curl -X GET "https://api.zenflow.com.ar/api/v1/products/sku/SKU-001" \
  -H "X-API-Key: zenflow_live_your_key"
```

### Response

```json
{
  "success": true,
  "data": {
    "id": 100,
    "barcode": "7891234567890",
    "external_id": "SKU-001",
    "name": "Widget A",
    "category": "Electronics",
    "price": 29.99,
    "stock": [
      {
        "location_id": "LOC-A1",
        "quantity": 50
      }
    ]
  }
}
```

## Error Response

```json
{
  "success": false,
  "error": {
    "code": "not_found",
    "message": "Product not found"
  }
}
```
