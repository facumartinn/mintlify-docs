# Delete Webhook

Delete a webhook configuration.

<Note>Requires `write:webhooks` scope.</Note>

## Request

```bash
DELETE /api/v1/webhooks/{id}
```

### Path Parameters

<ParamField path="id" type="string" required>
  The webhook ID
</ParamField>

## Example

```bash
curl -X DELETE "https://api.zenflow.com.ar/api/v1/webhooks/wh_abc123" \
  -H "X-API-Key: zenflow_live_your_key"
```

### Response

```json
{
  "success": true,
  "data": {
    "message": "Webhook deleted successfully"
  }
}
```

## Error Response

```json
{
  "success": false,
  "error": {
    "code": "not_found",
    "message": "Webhook not found"
  }
}
```

<Warning>
  This action is permanent and cannot be undone. All delivery history for this
  webhook will also be deleted.
</Warning>
