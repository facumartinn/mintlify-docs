# List Webhooks

Retrieve all configured webhooks.

<Note>Requires `read:webhooks` scope.</Note>

## Request

```bash
GET /api/v1/webhooks
```

### Query Parameters

<ParamField query="active" type="boolean">
  Filter by active status
</ParamField>

## Example

```bash
curl -X GET "https://api.zenflow.com.ar/api/v1/webhooks" \
  -H "X-API-Key: zenflow_live_your_key"
```
