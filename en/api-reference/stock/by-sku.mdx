# Get Stock by SKU

Retrieve stock levels for a product using its SKU.

<Note>Requires `read:stock` scope.</Note>

## Request

```bash
GET /api/v1/stock/sku/{sku}
```

### Path Parameters

<ParamField path="sku" type="string" required>
  The product SKU (external_id)
</ParamField>

## Example

```bash
curl -X GET "https://api.zenflow.com.ar/api/v1/stock/sku/SKU-001" \
  -H "X-API-Key: zenflow_live_your_key"
```

### Response

```json
{
  "success": true,
  "data": {
    "product_id": 100,
    "sku": "SKU-001",
    "product_name": "Widget A",
    "total_quantity": 75,
    "locations": [
      {
        "location_id": "LOC-A1",
        "quantity": 50
      },
      {
        "location_id": "LOC-B1",
        "quantity": 25
      }
    ]
  }
}
```

## Error Response

```json
{
  "success": false,
  "error": {
    "code": "not_found",
    "message": "Product not found"
  }
}
```
